despre

Mate-Romania:

Mate-Romania.tech este un website fondat de Maxim Sabin pentru a ajuta elevii din clasele V-VIII din Romania cu formulele plictisitoare si repetitive din matematica cu algoritmi ce fac toata munca inutila si elimina timpul pierdut. Principalul scop al proiectului este incurajarea elevilor de a lucra mai multe probleme de performanta unice, dar si eliminarea "tocitului" la matematica. Alt scop ar fi dezvoltarea mea personala in HTML/CSS/JS, dar si alte framework-uri/platforme legate de cele mentionate. Pentru informatii de contact mergeti la sectiunea Contact .

Proiecte:

Proiectele de pe prima pagina sunt in majoritate programe (mai multe in curand!) realizate in HTML si JavaScript care automatizeaza mai multe calcule (nu promit ca sunt facute in cel mai eficient mod). Calculele sunt facute client-side, iar codul lor le puteti gasi in sursa pagini.

Gazduire web, donatii si altele:

Website-ul este gazduit pe GitHub Pages, gratuit. Domeniul este de la get.tech, si el gratuit cu ajutorul Pack-ului oferit de GitHub. Mai multe servicii/tehnologii folosite puteti gasi in README-ul de pe pagina de GitHub a site-ului. Pentru donatii va rugam contactati-ne. Donatiile nu sunt obligatorii dar sunt un gest bine-venit. Toate donatiile vor merge catre dezvoltarea site-ului si a altor programe de pe el.
